# Sonic-Snap

A simple clone of the mobile game "Marvel-Snap", featuring characters from "Sonic the Hedgehog".

## Installation

* Install UnityHub here: https://docs.unity3d.com/2020.1/Documentation/Manual/GettingStartedInstallingHub.html
* In UnityHub, install UnityEditor version 2021.3.16f1
* Add the folder of this cloned repository to UnityHub's project list, set it's version to the one installed in the previous step
* Make sure to set Unity's default editor to your preferred code editor
